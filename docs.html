Setup

With Vite:

Make a vite.js project 

        npm create vite@latest

Install React Three 

        npm install three @react-three/fiber

Add a style file in src/index.css 

        html,
        body,
        #root
        {
          position: fixed;
          top: 0;
          left: 0;
          width: 100%;
          height: 100%;
          overflow: hidden;
        }

Create your cool App inside a file called src/App.jsx 

        import { Canvas } from "@react-three/fiber";

        function App() {
          return (
            <>
              <Canvas>
                <mesh>
                  <torusKnotGeometry />
                  <meshNormalMaterial />
                </mesh>
              </Canvas>
            </>
          );
        }

        export default App;

Enjoy! 

        npm install
        npm run dev

From Scratch:

Make a node.js project 

        mkdir CoolProject
        cd CoolProject
        npm init -y

Install React and React Three 

        npm install react react-dom react-scripts three @react-three/fiber

Add dev and build scripts to package.json 

        "scripts" : {
          "dev": "react-scripts start",
          "build": "react-scripts build"
        }

Make a public folder with an index.html 


        <!-- just write "html:5" if you are using vscode -->

        <!DOCTYPE html>
        <html lang="en">
        <head>
          <meta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>Document&/title&
        </head>
        <body>
          
        </body>
        </html>

Add a div to the body to contain our React app 

        <body>
          <div id="root"/>
        </body>

Make an src folder with an index.js 

        import ReactDOM from "react-dom/client";
        import App from "./App";

        ReactDOM.createRoot(document.getElementById("root")).render(<App />);

Add a style file in src/index.css 

        html,
        body,
        #root
        {
          position: fixed;
          top: 0;
          left: 0;
          width: 100%;
          height: 100%;
          overflow: hidden;
        }
