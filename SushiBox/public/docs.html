Scope

  • Sushi Piece

    This can be done by subdividing a cube and making an inset that is later extruded several times. Materials with a base color are then assigned, also an array modifier that is applied when exporting.

  • Plate

    The plate can be modeled from a plane, extruding it to have some thickness, then the inset tool can be used to make a border. Extrude the center down and assign a material with a dark base color to complete the plate.
